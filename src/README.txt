--- README for ARITHMETICHELPER ---
      Written by Harry Sauers

    * Run "ArithmeticHelper.jar" to run the program.
    * If needed, alter "scores.txt" to reflect test case data.
    * Follow the on-screen instructions to practice your arithmetic.
    * Make sure to use standard integer format for putting in your answers.
    * Division is integer division, so 1/3 = 0 and 6/5 = 1.
    * Scoring is based on the number of questions you answered correctly
       and the ratio of correct to incorrect answers.
